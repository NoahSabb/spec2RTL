Spec2RTL translates hardware specifications into synthesizable Verilog using a spec clarification + RTL generation pipeline. Two backends are available: Claude Sonnet (cloud) and a fine-tuned Qwen 32B RL model (local vLLM).

The RL model (Noahsabb/spec2rtl-qwen32b-lora-rl-v2) was trained with GRPO on the CVDP benchmark, achieving 58.97% pass rate — beating the Claude standalone baseline of 55.13%.
